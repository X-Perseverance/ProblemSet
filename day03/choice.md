### 5. 下列main()函数执行后的结果为()
```c
int func()
{
	int i, j, k=0;
	for(i=0,j=-1; j=0; i++,j++)
	{
		k++;
	}
	return k;
}

int main()
{
	cout << (func());
}
```

- [ ] A. -1
- [x] B. 0
- [ ] C. 1
- [ ] D. 2

> **【分析】**<br>
> &#160; &#160; &#160; &#160; `j=0` 这条赋值语句结果是零值，即为逻辑假，所以循环并不会进入，直接返回0。
